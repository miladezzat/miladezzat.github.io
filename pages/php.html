What is PHP?

Like in PHP Docs, PHP (recursive acronym for PHP: Hypertext Preprocessor) is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.

How is PHP Executed?

  1. it is received as HTML to be displayed by a browser
  2. php can be used as a general purpose programming language to write programs
  3. closing `?>` tag not required but recommended to use
  4. php ignores whitespace(tabes, spaces, new lines).
  5. unlike many other language, php not always case-sensitive, but it's best practice to use casing, Keywords are not case sensitive in PHP. As a convention, use the standard casing.
  6. `;` is required in the end of statement
